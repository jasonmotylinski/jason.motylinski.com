B) CSS Files and Structure - top

This theme imports seven CSS files.

  1. bootstrap.css
  2. font-awesome.min.css
  3. font-tello.css
  4. magnific-popup.css
  5. animation.css
  6. style.css
  7. yourcss.css

To add your own style you can modify the yourcss.css

	/* start wrapper page section */

	some code

	/* === Header Section  === */
	
	some code
	
	/* === Start  wrapper for navigation section ===*/
	
	some code

        /* === Global CLass ===*/
  
        some code
	
	/* === Footer === */
	
	some code
	
	etc, etc.

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

C) JavaScript - top

This theme imports eight Javascript files.

  1. jQuery.js
  2. pace.js
  3. bootstrap.js
  4. jquery.autofix_anything.js
  5. jquery.mixitup.min.js
  6. gmap3.min.js
  7. jquery.magnific-popup.min.js
  8. app.js
  1. jQuery is a Javascript library that greatly reduces the amount of code that you must write
  2. Pace is a Javascript and CSS library to automatically add beautiful progress and activity indicators for page loads and ajax navigation
  3. Both bootstrap.js and bootstrap.min.js contain all plugins in a single file. Include only one.
  4. jquery.autofix_anything is a plugin that will let you automatically fix position of any container on your website with one JS call
  5. MixItUp is a jQuery filter and sort plugin
  6. gmap3 is a JQuery plugin to create google maps with advanced features
  7. magnific-popup is Light and open source responsive lightbox plugin with focus on performance, for jQuery
  8. app.js is all configuration code in this template
